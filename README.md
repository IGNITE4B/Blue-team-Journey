# Blue-team-Journey
Blue Team Journey is a cybersecurity repository focused on defensive security, threat detection, incident response, and SOC operations. It contains practical labs, blue team tools, detection techniques, and hands-on learning resources for improving real-world cyber defense skills.

---

## Current Focus

- SC-900 fundamentals
- Linux administration
- Networking basics
- SIEM concepts
- Microsoft Sentinel & KQL
- Security operations workflows
